# HTML Starterkit
Basic frontend template for projects using HTML, CSS, Javascript.

## Basic Usage

``` bash
git clone https://github.com/fahmadrifki/html-starterkit.git YOUR_PROJECT_NAME
cd YOUR_PROJECT_NAME
git remote remove origin
```

*delete folder: .git
